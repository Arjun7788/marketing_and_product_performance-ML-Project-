 Marketing & Product Performance Analysis using Machine Learning

 Overview

This project applies supervised machine learning techniques to analyze and predict product sales based on marketing data.

The study answers 7 research questions covering:

* Model performance
* Feature importance
* Preprocessing impact
* Robustness
* Final model selection

---

##  Project Structure

* `notebooks/` → Jupyter/Kaggle notebooks for each research question
* `results/` → all screenshots of results  


---

##  Methods Used

* Linear Regression
* Decision Tree
* Random Forest
* XGBoost
* Support Vector Regression

Evaluation Metrics:

* MAE
* RMSE
* R² Score

---

##  Key Findings

* Ensemble models (Random Forest, XGBoost) performed best
* Marketing features significantly influence sales
* Proper preprocessing improves performance
* Models show stable performance under different conditions

---
